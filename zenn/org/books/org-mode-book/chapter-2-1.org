#+TITLE: 2.1. アウトライン操作の基本
#+DESCRIPTION: description
#+STARTUP: fold
#+OPTIONS: toc:nil
* 見出しの作成と編集

- M-RET: 同レベルの見出し作成
- M-S-RET: TODOつき見出し作成
- C-RET: 見出しの下に新規見出し

* 見出しの移動

- M-UP / M-DOWN: 見出しの上下移動
- M-LEFT / M-RIGHT: 見出しの昇格/降格
- M-S-LEFT / M-S-RIGHT: サブツリーごと昇格/降格

* 折りたたみ（Visibility Cycling）

- TAB: 現在の見出しをサイクル（折り畳み→子を表示→全展開）
- S-TAB: バッファ全体をサイクル
- C-u C-u TAB: 全展開
- org-startup-folded の設定

* ナローイング

- C-x n s: サブツリーにナロー
- C-x n w: ナロー解除
- 集中編集のためのテクニック

* リスト操作

- M-RET: リスト項目の追加
- M-UP/DOWN: リスト項目の移動
- C-c C-c: チェックボックスのトグル
- C-c C-*: リスト→見出し変換

* ナビゲーション

- C-c C-n / C-c C-p: 次/前の見出し
- C-c C-f / C-c C-b: 同レベルの次/前
- C-c C-u: 親の見出しへ
- org-goto （C-c C-j）: インクリメンタル見出し移動
- imenu 連携
